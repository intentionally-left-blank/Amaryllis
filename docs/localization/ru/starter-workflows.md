# Стартовые workflow

### Сценарий 1

Локальная активация:

1. Проверить профиль оборудования.
2. Выбрать пакет модели.
3. Запустить активацию и smoke-check.

### Сценарий 2

Релизный контроль:

1. Запустить обязательные release gate проверки.
2. Проверить отчеты качества и adoption.
3. Подготовить rollback-план перед публикацией.
